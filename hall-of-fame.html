How the Daily Run works

Loop Runner's Daily Run is a separate game mode that runs on a fixed, shared random seed. Every player who attempts the Daily Run on a given day faces the same enemy spawns in the same order at the same speed — the same power-up drops in the same places at the same timestamps, the same boss archetypes at the same milestones. The seed regenerates at 00:00 UTC and the previous day's scores are sealed and archived.

The point is fairness. Normal mode uses randomised spawns, which means raw scores between two Normal-mode players are only loosely comparable — one player might have drawn a quiet spawn pattern at a critical moment, another might have been swarmed at the same score. The Daily Run removes that variance. If you and another player both attempt today's seed and they score higher, they played it better. There is no other explanation.

What this archive is

Every completed day, the top three Daily Run scores become a permanent record. They live here in the Hall of Fame, ordered newest-first, with gold/silver/bronze medals indicating finishing position. Each card shows the day (with "Today" highlighted), the player's display name, their score, and their country code if available. Empty rows in a card mean fewer than three players posted Daily Run scores that day — historically this is rare for recent days but more common for the earliest archived dates, before the game had its current player base.

The archive is computed live from the leaderboard database — it isn't a cached file. When you load this page, your browser fetches the most recent Daily Run submissions (currently the last 500), buckets them by submission date in UTC, and renders the top three per bucket. That means new podiums appear here within seconds of the day ending, and we don't need a scheduled job to roll the archive forward.

How scores are validated

A Daily Run score is accepted into the archive when:

  • It was submitted in Daily mode (Normal-mode scores never appear in the Hall of Fame, no matter how high).
  • The submitting client passed basic sanity checks — sane time-to-score, plausible kill-per-minute pacing, no impossible jumps in the score timeline.
  • The display name doesn't violate our Terms of Use. Names containing hate speech, real-world personal data, or harassment are removed; the score still counts, but the entry won't appear with that name.

We treat the leaderboard as a record of skill, not a personal data store — see the Privacy Policy for what we store and what we don't.

Why the Daily Run is the most realistic path to a podium

The all-time worldwide leaderboard rewards the best run ever played. Posting on it requires a single outlier session against the entire history of the game. The Daily Run podium, by contrast, only requires beating the other people who attempted today's specific seed. On most days that's a much smaller field.

Practically speaking: a high Normal-mode score takes hours of practice and the right run. A Daily Run podium often takes three or four attempts on the same day, because route knowledge accumulates inside a single session. The first attempt is reconnaissance, the second sets the route, the third is for the score. The Strategy Guide covers Daily-specific tactics in detail.

What the country codes mean

The two-letter code next to a name is the player's country at the time of submission, derived from a public IP-to-country lookup. The lookup happens on the player's own browser; only the resolved code is sent to our database. We don't store IP addresses and we don't collect city-level location. A missing or "XX" code means geolocation was blocked or returned no answer — usually a VPN, a privacy extension, or a regional ISP that the lookup service doesn't classify.
